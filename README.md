## Bill of Materials (BOM)
Here's everything you'll need to build this:

4x Cherry MX Switches
4x Keycaps of your choice
6x SK6812 MINI-E LEDs
1x EC11 Rotary Encoder
1x Seeed XIAO RP2040
8x M3 Nut and Bolt, 1.5cm,
1 Case 
1 Custom 

## Firmware
This uses KMK firmware in Python

The rotary encoder changes volume
Neopixels to backlight 
Keys for WASD for now but can be modified
